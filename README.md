# cs624-pe-akshita

# Input

The application does not require any input from the user. It displays predefined information including my name, degree program, and school name when the application starts.

# Process

The React Native application uses one View component to hold three Text components. A StyleSheet is used to apply formatting such as a yellow background, centered alignment, and bold text. When the application is launched, React Native renders the information on the screen automatically.

# Output

The application displays my name, "Akshita Modi," my degree program, "Master of Science in Computer Science," and my school, "City University of Seattle," centered on a yellow background.
